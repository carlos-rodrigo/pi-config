# Themes

## catppuccin-macchiato

A Pi theme using the [Catppuccin Macchiato](https://github.com/catppuccin/catppuccin) color palette — a warm dark theme with pastel accents.

### Palette mapping

| Role | Catppuccin Color | Hex |
|---|---|---|
| Accent / Keywords | Mauve | `#c6a0f6` |
| Success / Strings | Green | `#a6da95` |
| Error | Red | `#ed8796` |
| Warning / Types | Yellow | `#eed49f` |
| Headings | Peach | `#f5a97f` |
| Links / Functions | Blue | `#8aadf4` |
| Inline code / Bullets | Teal | `#8bd5ca` |
| Variables | Flamingo | `#f0c6c6` |
| Operators | Sky | `#91d7e3` |
| Numbers | Peach | `#f5a97f` |
| Muted text | Overlay1 | `#8087a2` |
| Dim text | Overlay0 | `#6e738d` |
| Backgrounds | Base / Mantle / Crust | `#24273a` / `#1e2030` / `#181926` |

### Thinking level borders

| Level | Color |
|---|---|
| Off | Surface1 `#494d64` |
| Minimal | Surface2 `#5b6078` |
| Low | Sapphire `#7dc4e4` |
| Medium | Blue `#8aadf4` |
| High | Mauve `#c6a0f6` |
| Xhigh | Pink `#f5bde6` |

### Usage

Select via `/settings` → theme → `catppuccin-macchiato`, or set in `~/.pi/agent/settings.json`:

```json
{
  "theme": "catppuccin-macchiato"
}
```
